# 엄만달

교대근무 간호사를 위한 근무표 자동생성 웹앱 — 엄마의 한 달을 1분 만에

## 사용 주소

https://songchoseungdal.github.io/ummandal/

## 특징

- **근무표 자동생성**: 인원만 등록하면 규칙에 맞는 근무표 초안이 바로 완성됩니다.
- **공평 통계**: 나이트·주말·오프 횟수를 한눈에 보여줘 형평성 있게 배정합니다.
- **이미지 공유**: 완성된 근무표를 이미지로 저장해 손쉽게 공유할 수 있습니다.
- **오프라인 PWA**: 앱처럼 설치해 인터넷 없이도 사용할 수 있습니다.
- **기기 간 동기화**: 서버 동기화로 여러 기기에서 같은 근무표를 이어서 편집합니다.
